 Next Word Generator (LSTM)

A clean and minimal Next Word Prediction Web App built using:

🧠 LSTM (TensorFlow / Keras)
🌐 Streamlit
🎨 Custom pastel UI
🚀 Features
Predicts next 10 words
Smooth & minimal UI (glassmorphism)
Fast inference using trained LSTM model
🖥️ Demo

Type a sentence and the model generates the next sequence of words.

📁 Project Structure
app.py
lstm_model.h5
tokenizer.pkl
max_len.pkl
requirements.txt
⚙️ Installation
git clone https://github.com/your-username/next-word-generator.git
cd next-word-generator
pip install -r requirements.txt
▶️ Run App
streamlit run app.py
⚠️ Note

Model files (.h5, .pkl) may not be included due to size.
Add them manually before running.
